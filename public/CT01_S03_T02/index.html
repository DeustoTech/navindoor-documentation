T02 Introduction to connections class

Operaciones básicas con la clase Level

Crearemos un nivel básico para ver las funciones mas básicas que podemos realizar apartir de este objeto.

clear;
n1 = node([10 10]) ; n2 = node([40 10]) ; n3 = node([40 40]) ; n4 = node([10 40]) ;
w1 = wall([n1 n2]) ; w2 = wall([n2 n3]) ; w3 = wall([n3 n4]) ; w4 = wall([n4 n1]) ;
% creamos un nivel con los nodes y paredes creados
ilevel = level;
ilevel.nodes = [n1 n2 n3 n4];
ilevel.walls = [w1 w2 w3 w4];
ilevel.dimensions = [50 50];
ilevel


ilevel =

Una de las operaciones más básicas podemos ver gráficamente este objeto

clf
